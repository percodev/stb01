# Конфигурация
### Запрос:
```json
{
    "get":"exdev","exdev":{}
}
```

### Установка:
```json
{
    "set":"exdev", 
    "exdev":{
       "mode":"slave",
       "passage_time":"7000"
    }
}
```

### Ответ:
```json
{
    "answer":{"exdev":"ok"},
    "exdev":{
       "mode":"slave",
       "passage_time":"7000"
    }
}
```
